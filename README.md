* [Optimized interpreter](https://github.com/kaixindeken/Brainfuck-Rust/blob/main/src/main_ir.rs)

* [Optimized interpreter with JIT](https://github.com/kaixindeken/Brainfuck-Rust/blob/main/src/main_jit.rs)
